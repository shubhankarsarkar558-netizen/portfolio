## 🌐 Deployment

Hosted on **Vercel** with custom domain via **Cloudflare DNS**.

```
soumyadip.ucoder.in
      ↓
Cloudflare DNS (CNAME → cname.vercel-dns.com)
      ↓
Vercel (Next.js build)
```

**To deploy your fork:**
1. Push to GitHub
2. Import on [vercel.com](https://vercel.com)
3. Settings → Domains → Add `your-subdomain.yourdomain.com`
4. Add `CNAME` in Cloudflare → target `cname.vercel-dns.com` → Proxy **OFF**
